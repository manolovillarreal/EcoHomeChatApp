EcoHome Chat

EcoHome Chat corresponde al desarrollo del sistema de mensajería interna para EcoHome Store. El proyecto fue construido como caso académico para demostrar integración entre frontend, autenticación JWT, WebSockets y persistencia en PostgreSQL.

Funcionalidades principales

  • Inicio de sesión usando JWT emitido por un servicio de autenticación externo.
  • Chat global en tiempo real para todos los usuarios conectados.
  • Listado de usuarios en línea con presencia activa.
  • Mensajería privada uno a uno entre usuarios conectados.
  • Persistencia de mensajes dentro del esquema chat en PostgreSQL.
  • Historial reciente para chat global y conversaciones privadas.
  • Frontend desplegable dentro del backend mediante archivos estáticos en public/.

Arquitectura

chat-app Frontend en React + Vite. Maneja login, vista de chat, presencia, mensajes privados y despliegue del build.
chat-service Backend en Node.js + Express + Socket.IO. Valida JWT, coordina eventos en tiempo real y persiste mensajes.
PostgreSQL Almacena mensajes en el esquema chat con soporte para mensajes globales y privados.
Auth Service Servicio externo encargado de autenticar usuarios y emitir el token JWT consumido por frontend y chat.

Eventos principales

message-history            -> historial de chat global
new-message                -> mensaje global
users:online               -> usuarios conectados
private-message            -> mensaje privado 1:1
private-history            -> solicitud de historial privado
private-message-history    -> historial privado entre dos usuarios

Rutas y acceso

  • GET /health - comprobación de estado del microservicio.
  • / - frontend del chat servido por Express.
  • /login - ruta de acceso de la SPA.
  • /chat - interfaz principal del chat.
  • Handshake Socket.IO autenticado con auth: { token }.

Documentación y despliegue

El frontend se construye en chat-app y puede copiarse hacia chat-service/public mediante el comando npm run deploy. El backend queda preparado para servir la aplicación compilada junto con la API y los sockets. 

chat-app
npm run deploy

chat-service
npm install && npm run build
npm start
